DataFlow-X 是什么?

DataFlow-X 是用于实时流计算的平台,核心是通过工作流描述构建分布式流计算应用程序, DFX是一套完整的解决方案, 主要包括:

  • 开发工具: webUI, 定义streamSql或streamETL任务
  • 运行时: 基于 Apache Spark2.x 及 Apache Flink1.5+, 依赖 Apache Hadoop 和 Yarn
  • Pipline插件扩展: java8, 按需实现 Source => Transform => Sink
  • 基础运维: 在 DFX webUI 完成任务创建后一键部署上线, 无需人工部署, 整个分布式程序由DFX进行管理
  • 运行分析: 在 DFX webUI 上面即可看到任务运行情况, 不再需要打隧道查看Yarn任务

# 设计原则

  • 利用插件系统分离 配置与实现
  • 约定优于配置, 按照统一的约定进行应用开发, 从开发工具到目录结构.

# 特性

  • 快速完成实时计算任务: StreamSql
  • 灵活性: 可简单自定义 pipeline节点插件
  • 便捷部署: 自动重启机制, 一键上线至Yarn
  • 简单运维: 日志监控

# 什么样的场景适合用DFX

目前DFX定位主要是 实时ETL数据处理能力 及 实时流计算, 基于两大能力, DFX有自身的特定应用场景, 主要如下:

  • 快速实验流计算
  • 管理流计算任务

What is DataFlow-X?

DataFlow-X is a platform for real-time flow computing, the core is to build distributed flow computing applications through workflow description., DFX is a complete solution, mainly including:

  • Development Tools: webUI, Define stream Sql or stream ETL task
  • RunTime: based on Apache Spark 2.x and Apache Flink 1.5+ , rely Apache Hadoop and Yarn
  • Pipline Plugin Extension: java8, On demand Source => Transform => Sink
  • Basic Operation And Maintenance: in DFX webUI One-click deployment and online after completing task creation, No manual deployment, The entire distributed program is managed by DFX
  • Run Analysis: You can see the task running status on the DFX webUI, no longer need to tunnel to view Yarn tasks

# Design Principles

  • Separation using plug-in system Configuration and implementation
  • Convention is better than configuration, according to a unified convention for application development, from development tools to directory structure.

# Characteristic

  • Quickly complete real-time computing tasks: Stream Sql
  • Flexibility: Easy to customize pipeline node plugin
  • Easy Deployment: Automatic restart mechanism, one-click online to Yarn
  • Simple Operation And Maintenance: Log monitoring

# What kind of scene is suitable for DFX

The current DFX positioning is mainly Real-time ETL data processing capability and Real-time streaming computing, Based on these two capabilities, DFX has its own specific application scenarios, mainly as follows:

  • Fast experimental flow calculation
  • Manage stream computing tasks

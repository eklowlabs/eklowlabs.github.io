REST Service

In DFX, Restful style interface will be used:

  • Unified management of back-end service configuration;
  • Package exception handling, timeout retry, improve system stability;
  • Encapsulated logs for easy online problem tracking;
  • Abstract request protocol, data format and data encoding, standardized unified interface;
